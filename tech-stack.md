# Tech Stack

## Frontend
- **React + TypeScript + Vite**
- **Tailwind CSS**
- **React Router**

## Backend
- **Node.js + Express + TypeScript**

## Database
- **PostgreSQL + Prisma**

## Auth
- **Database sessions** — session tokens stored in the database, referenced via a session cookie

## AI
- **Claude API** (claude-sonnet-4-6)

## Email
- **SendGrid** or **Mailgun** — inbound webhooks to Express for receiving emails, outbound for replies

## Deployment
- **Docker** (containerized frontend + backend)
- Cloud provider: **AWS**, **Railway**, or **Fly.io**
